# AgentMail

A basic Golang CLI application.

## Installation

```bash
go build -o agentmail
```

## Usage

```bash
# Show help
./agentmail help

# Show version
./agentmail version

# Say hello
./agentmail hello
./agentmail hello --name Alice
```

## Commands

- `version` - Display version information
- `hello` - Say hello with optional name flag
- `help` - Show usage information
